# 플코 프론트엔드 과제

### 1. 소요 시간
  약 6시간 이내


### 2. directory 구조 설명
    src 폴더
    - assets : css, icon 이미지 폴더
    - components
      - Pagination.tsx : 페이지네이션 
      - Player.tsx : 플레이어 
      - PlayerList.tsx : 플레이어 리스트 
      - Filter.tsx : 필터 
    - constants
      - types.ts : 타입 공통 변수
    - pages 
      - Home.tsx : 홈

    - 기본
      - App.tsx
      - index.tsx
      - player.json : 플레이어 더미 데이터 (플레이어 1명 추가)
